Componentes Important

Partes de un sistema computacional

Son componentes o equipos físicos que intervienen en un sistema de computacional.

Es un elemento básico para un sistema computacional, que se define como un equipo electrónico habilitado para interpretar y ejecutar ordenes de entrada y salida.

Es un componente lógico que no puede hacer nada sin que se le de la parte física para funcionar o interactuar con el ser humano. Ejemplo: Sistemas operativos (Linux, windows, solaris, etc.)

Son personas que tiene interactuar con el Hardware Y EL Software para la ejecución y intervención.
